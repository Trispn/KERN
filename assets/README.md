# KERN Assets

This directory contains official media assets for the KERN programming language.

## Contents

### Vector (SVG) - Recommended
- **[kern_logo.svg](kern_logo.svg)**: The official KERN logo. Scalable vector graphics.
- **[kern_file_icon.svg](kern_file_icon.svg)**: Icon for KERN source code files (`.kern`).
- **[kern_bytecode_icon.svg](kern_bytecode_icon.svg)**: Icon for KERN bytecode files (`.kbc`).
- **[kern_banner.svg](kern_banner.svg)**: Social media and repository banner image.

### Raster (PNG)
- **kern_logo.png**: High-res raster export.
- **kern_file_icon.png**: Raster file icon.
- **kern_bytecode_icon.png**: Raster bytecode icon.
- **kern_banner.png**: Raster banner.

## Usage

These assets are free to use for any KERN-related project, documentation, or tooling.
