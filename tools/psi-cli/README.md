PSI CLI (prototype)

This crate provides a minimal prototype REPL for PSI that:

- Accepts interactive commands (`--interactive`) or a batch file (`--batch`).
- Generates a simple KERN file `demo/psi_generated.kern` when `generate` is used.
- Calls the existing `kernc` compiler CLI to build and run the generated bytecode.
- Emits simple language stubs for `rust` and `python` targets in `demo/`.

Usage examples:

```powershell
# interactive
cargo run -p psi_cli -- --interactive

# batch
cargo run -p psi_cli -- --batch tasks.txt

# load a brain
cargo run -p psi_cli -- --load brain.json
```
